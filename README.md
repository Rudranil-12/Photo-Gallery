## 📸 Photo Gallery App

A modern and responsive Photo Gallery Web App built using React.js and Tailwind CSS. This project displays a collection of images in a clean grid layout with a smooth UI and filtering functionality.

---

## 🚀 Features

📷 Responsive image gallery layout
🔍 Filter images by author name
❤️ Like/Favorite button UI
🌙 Dark-themed modern UI
⚡ Fast performance using React
🎨 Styled with Tailwind CSS

---

## 🛠️ Tech Stack

Frontend: React.js
Styling: Tailwind CSS
Deployment: Vercel

---

## 🎯 How It Works

- The app fetches and displays images in a grid layout
- Users can filter images by author name
- Each card includes:
- Image preview
- Author name
- Like button

---

- Modern gallery systems like this are typically responsive and customizable, maintaining image ratios and grid alignment for better UX
